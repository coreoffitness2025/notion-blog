## 현재 상태 진단

### 잘 되어 있는 것

- NutritionInformation, HowTo JSON-LD 구조화 데이터 우수
- hreflang 한/영 전 페이지 설정
- Google + Naver 사이트 검증 완료
- Next.js SSR로 크롤링 친화적

### 즉시 수정 필요
1. `/posts` 페이지 og:title이 메인과 동일 ("CoreVia - 진짜 온라인 PT") -> 고유값 필요
2. `/posts` og:url이 메인 URL -> `/posts`여야 함
3. 블로그에 Article/BlogPosting JSON-LD 스키마 없음
4. 블로그 포스트 전부 2025년 9월 -> 신선도 문제

### 핵심 과제
- 11,500페이지 중 실제 인덱싱 비율 불명 (Search Console 확인 필요)
- 영양 DB 페이지가 thin content로 판정될 리스크
- 네이버 SEO 미착수

## Phase 2: 콘텐츠 품질 강화 (1-2주)

### 2-1. 영양 DB 페이지 thin content 탈출

현재 영양 페이지가 칼로리/영양소 테이블만 있으면 **Google이 thin content로 판정**할 위험이 높음. 11,344페이지 전부가 품질 낮으면 사이트 전체 순위 하락(HCU).

**전략: 2단계 접근**

**Tier 1 (상위 500개)**: 검색량 높은 음식 (닭가슴살, 계란, 바나나, 고구마 등)
- 영양소 테이블 + 운동 목적별 활용법 + 대체 식품 + FAQ 2-3개
- FAQ Schema 적용 -> 리치 스니펫 노출 가능
- 페이지당 고유 콘텐츠 500자+

**Tier 2 (나머지 10,844개)**: 일단 noindex 고려
- 인덱싱 비율 확인 후 결정
- 품질 보강 가능한 것부터 점진적으로 인덱싱
- 모든 페이지를 동시에 인덱싱하면 crawl budget 낭비

### 2-2. Internal Linking 강화

현재 각 페이지가 고립(orphan page) 상태일 가능성. 연결 필요:

```
블로그 "컷팅 가이드" -> 영양 DB "닭가슴살", "고구마", "브로콜리"
운동 "스쿼트" -> 영양 "단백질 많은 음식" -> 블로그 "근비대 원칙"
영양 "닭가슴살" -> 관련: "계란", "연어", "두부" + 블로그 "단백질 타이밍"
```

**실행**: 각 페이지 하단에 "관련 콘텐츠" 섹션 자동 생성 (같은 카테고리/근육군/영양소 기준 5-8개 링크)

### 2-3. E-E-A-T 강화

피트니스/영양은 **YMYL(Your Money Your Life)** 카테고리. Google이 가장 엄격하게 평가.

| 요소 | 실행 |
|------|------|
| Experience | 앱 사용자 데이터 활용 ("CoreVia 사용자 78%가 이 음식을 점심에 선택") |
| Expertise | 저자 프로필 페이지 생성 (/about/team/[name]), 자격증/경력 명시 |
| Authority | 과학 논문 인용, 식약처/USDA 데이터 출처 표시 |
| Trust | "최종 검토: 2026-04-08" 날짜 표시, 의학적 면책 조항 |

## Phase 3: 인덱싱 가속 (2-4주)

### 3-1. Search Console 데이터 확인 (최우선)
- 인덱싱된 페이지 수 확인 (Coverage 리포트)
- "Discovered - currently not indexed" 비율
- 크롤링 에러
- 모바일 사용성 문제

### 3-2. 점진적 인덱싱 전략
```
1주차: 핵심 224개 (운동 198 + 블로그 26) 완전 인덱싱 확인
2주차: 영양 DB 상위 1,000개 sitemap 추가
3주차: 영양 DB 2,000-4,000개 추가
4주차: 인덱싱 비율 90%+ 달성 확인 후 다음 배치
```

### 3-3. IndexNow API 자동화
Bing + Naver 대응. 페이지 업데이트 시 자동 호출:
```javascript
// API route 또는 빌드 후 스크립트
fetch('https://api.indexnow.org/IndexNow', {
  method: 'POST',
  body: JSON.stringify({
    host: 'coreviafitness.com',
    key: 'YOUR_KEY',
    urlList: updatedUrls // 최대 10,000개
  })
})
```

### 4-3. 네이버 전략 - 한국 검색 46%

**네이버 블로그 운영 (필수)**:
- 주 2-3회 포스팅 (최소 3개월 지속 -> C-Rank 신뢰도 축적)
- 마케팅 에이전트 파이프라인으로 자동 생성 가능
- 블로그 -> coreviafitness.com -> 앱 설치 퍼널

**네이버 노출 팁**:
- 비교 콘텐츠는 **표(table) 형태**로 -> 스마트블록 노출 확률 상승
- 시리즈물 > 단발성 포스트
- 인포그래픽/데이터 시각자료 포함

- `VideoAgent` -> 30-60초 숏폼 스크립트
